# Python Project - Pedra, Papel e Tesoura

O seguinte projeto é um jogo para 2 jogadores, o clássico pedra, papel ou tesoura porém desenvolvido usando Python.

## Funcionamento

Passo 1: Ao iniciar o jogo os jogadores terão que informar seus nomes.<br>
Passo 2: Após isso o jogo irá questionar seu conhecimento das regras, caso você já conheça marque a opção que 
se encaixe e continue, caso não conheça marque a opção que não conhece e ele retornará com as regras.<br>
Passo 3: Agora o jogo começa, ambos terão que escolher entre as opções, e após o jogo irá mostrar quem ganhou o round e perguntar se desejam jogar de novo.<br>
Passo 4: Caso tenha marcado que não deseja jogar novamente, o jogo irá mostrar o placar final entre os jogadores (Vitórias e Empates) e irá encerrar o código, sendo necessário roda-lo de novo para jogar.
